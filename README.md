# Tender Scout

Intelligent tender matching for Swiss SMEs. Aggregates public contracts from SIMAP, TED, and EU portals.

## Quick Start

Open `index.html` in a browser. No build step required.

## Stack

- Pure HTML/CSS/JS
- No dependencies
- Mobile-first design

## Files

| File | Purpose |
|------|---------|
| `index.html` | Landing page |
| `VISION.md` | Product vision |
| `DESIGNGUIDE.md` | Design system |
| `MARKET.md` | Market research |
